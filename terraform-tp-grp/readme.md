# TP — Conception et déploiement d’une infrastructure AWS avec Terraform

## Contexte
Une startup souhaite déployer une application web dans le cloud. L’infrastructure attendue doit
être sécurisée, modulaire, reproductible et adaptable à plusieurs environnements. Vous êtes
chargés de concevoir puis d’implémenter cette infrastructure avec Terraform.
